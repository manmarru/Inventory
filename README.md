# Inventory
[시연영상](https://youtu.be/K9rdjz1qtuw)

i : 인벤토리 열기  
1~9 : 코드에 맞는 아이템 추가  
우클릭: 아이템 사용  
휠클릭: 아이템 정렬잠금/해제  
활성화된 버튼: 정렬, 펫슬롯  

아이템을 좌클릭으로 끌어서 다른 아이템 슬롯에 놓거나 아이템간의 위치를 바꿀 수 있습니다.  
펫슬롯 버튼으로 아이템을 끌어다 놓으면 펫/플레이어 인벤토리로 아이템이 옮겨집니다.  
인벤토리 위쪽을 좌클릭으로 끌어서 ui창을 이동할 수 있습니다.  



(디버깅용 LURD_TEST 비활성화 상태)
